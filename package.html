yum과 dnf

  • RHEL에서 패키지관리자는 yum 및 dnf
  • RHEL7 버전까지는 yum 사용, RHEL8 버전부터 dnf 사용
  • yum은 파이썬2 기반, dnf는 파이썬3 기반
  • RHEL8 이후 버전의 yum은 내부적으로 dnf 호출

yum

yum 명령어 표

명령 레포지토리 패키지 그룹
목록 repolist list grouplist

검색 - search -
의존성 정보 - deplist -

설치 - install groupinstall
제거 - remove groupremove
